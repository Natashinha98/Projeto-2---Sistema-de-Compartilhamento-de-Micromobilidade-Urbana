## Sobre o projeto

Este projeto apresenta a modelagem UML de um sistema de compartilhamento de micromobilidade urbana, voltado ao uso de bicicletas e patinetes elétricos em ambiente urbano.

O sistema permite que usuários consultem veículos disponíveis, realizem reservas, iniciem viagens por QR Code, finalizem viagens, efetuem pagamentos e consultem informações relacionadas ao uso do serviço. Também contempla funcionalidades administrativas e operacionais, como gerenciamento de frota, controle de zonas, monitoramento de veículos, geração de ordens de serviço e relatórios operacionais.
